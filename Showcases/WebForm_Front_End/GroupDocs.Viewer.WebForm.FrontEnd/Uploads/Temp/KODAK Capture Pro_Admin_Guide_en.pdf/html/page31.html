Double data entry  
Double Entry allows for two index entry operators to index the documents  
separately to improve the accuracy of the index data.  
When scanning is complete, the operator selects Index1 from the Index menu  
to indicate that the batch is ready for the first index operation.  
The first index operator enters the index values for the batch and for each  
document. When complete the index operator selects Index2 from the Index  
menu to indicate that the batch is ready for the second index operation.  
Double Entry setup  
To use Double Entry one or more batch or document index fields must  
configured for Double Entry.
