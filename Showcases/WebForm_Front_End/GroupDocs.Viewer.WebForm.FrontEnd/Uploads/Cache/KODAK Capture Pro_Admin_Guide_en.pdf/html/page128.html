Scanning an image  
for bar code setup  
Kodak Capture Pro Software allows you to separate pages and documents on  
your scanner with pages containing bar codes. It also allows you to create  
attachments to your documents using these bar code separators.  
Before you can use bar codes as separators, you must:  
•
•
•
Scan an image containing the bar codes into Capture Pro Software.  
Draw and select the bar code zones.  
Set up your job for bar code zone separation.  
Kodak Capture Pro Software allows you to scan bar code images directly into  
the Bar Code window for viewing, defining and editing.
