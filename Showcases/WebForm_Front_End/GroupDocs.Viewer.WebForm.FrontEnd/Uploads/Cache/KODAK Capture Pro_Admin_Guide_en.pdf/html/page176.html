User tab  
Users and user groups can be configured in Capture Pro Software to limit  
access to specific jobs and Capture Pro Software functions.  
Adding a new user  
1. Select File>User Setup.  
2
. Open the User tab.  
3
. Click New. The New User dialog box will be displayed.  
4
5
6
. Enter the name of the user in the User name field.  
. Enter a password for this user in the Password field.  
. Re-enter the password in the Confirm password field and click Create.  
This user is automatically added to the list.  
NOTE: The user can change this password by selecting File>Change  
Password.
