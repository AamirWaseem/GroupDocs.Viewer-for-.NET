Deleting a group  
1. Select File>User Setup and open the Group tab. The Group tab of the  
User Setup dialog box will be displayed.  
2
3
. Select the group that you want to delete from the Group name drop-down  
list.  
. Click the Delete icon. A confirmation box will be displayed. Click Yes to  
delete the group.  
NOTE: When you delete a group, the users in that group are removed  
from the group and returned to the Members: Not in Group list.  
Renaming a group
