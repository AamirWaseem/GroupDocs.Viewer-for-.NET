. When grouping by multi-page, select either All or Images as a way to

•
If you select All, all images in the Page, Document and Batch group  
selected will be in the output file.  
•
If you select Images, only the specified images in the Page, Document  
or Batch group selected will be in the output file. For example, if you  
selected Multi-Page, group by Document and specified an Output  
Range of 1, 10, then only the first and tenth image in each document  
will be in the output file. If a document contains less than 10 images,  
only the first image will appear in the output file.
