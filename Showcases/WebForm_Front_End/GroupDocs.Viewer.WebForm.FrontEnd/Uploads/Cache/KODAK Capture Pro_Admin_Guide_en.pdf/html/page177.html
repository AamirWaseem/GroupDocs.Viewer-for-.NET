Adding a user to a group  
1. Select File>User Setup and open the User tab. The User Setup dialog  
box will be displayed.  
2
3
4
. Select a user name from the User name drop-down list.  
. Select a group name from the Add to Group drop-down list.  
. Check Allow automatic login if you want to bypass the login function for  
Capture Pro Software and automatically login this user when Capture Pro  
Software is launched.  
NOTE: If you want to login as another user, select File>Logout and then  
login as the new user.  
5

Deleting a user  
1. Select File>User Setup and open the User tab. The User Setup dialog  
box will be displayed.  
2
3
4
5
. Select the user you want to delete from the User name drop-down list.  
. Click Delete. A confirmation box will be displayed.  
. Click Yes to complete the deletion.  
. Click OK.  
Resetting a password  
1. Select File>User Setup and open the User tab. The User Setup dialog  
box will be displayed.  
2
3
. Select the user you want to reset a password for from the drop-down list.  
. Click Reset Password. The Change Password - sysadmin dialog box will  
be displayed.  
4
5
6
. Enter a new password in the New password field.  
. Re-enter the password in the Confirm password field.  
. Click OK.  
NOTE: The user can change this password by selecting File>Change  
Password.
