Question: Why is the scanner Properties Event dialog box displayed when I  
press the scanner button?  
Answer: The operating system displays this dialog box when the button is  
pressed and none of the Kodak applications are running. Select an application  
listed in the dialog box to associate the scanner button to that application.  
Question: Can the buttons of non-Kodak scanners be configured to use  
shortcuts?  
Answer: No. The Button Manager, Shortcut Setup only supports Kodak  
scanners.
