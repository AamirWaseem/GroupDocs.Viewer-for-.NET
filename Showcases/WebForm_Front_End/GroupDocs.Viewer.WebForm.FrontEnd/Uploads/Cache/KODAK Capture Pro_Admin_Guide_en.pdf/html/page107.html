Font — from the drop-down list, select Large comic, Large cine, Large  
comic 180, Large cine 180, Small comic, Small cine, Small comic 180 or  
Small cine 180.  
Position — select a value to determine how far the printed information will  
appear from the leading edge of the document.  
Message Configuration — displays the Message Configuration dialog box for  
your scanner. This dialog box allows you to specify custom text to be included  
in your print string. A maximum of 20 characters is allowed. You can have up  
to six unique messages per string.  
Print string — enter a print string or click Setup to open the Formula  
Configuration dialog box and set up your print string using predefined values.  
New batch starts with counter equal to — enter the number you want to  
start a new batch with.
